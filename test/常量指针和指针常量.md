# 常量指针和指针常量

```cpp
	int a = 10;
	int b = 20;
	int c = 30;
	int d = 40;
	// 1. const 修饰指针：常量指针
	const int* p = &a; 
	p = &b; //指针的指向可以修改
	//*p = b; 指针指向的值不可以修改

	// 2. const 修饰常量：指针常量
	int* const p2 = &c;
	*p2 = 100; //指针指向的值可以修改
	//p2 = &b;   //指针的指向不可以改

	// 3. const即修饰指针，又修饰常量
	const int* const p3 = &d; //指针的指向和值都不可以修改

	// 4. 顶层const: 指针本身是个常量。等价于指针常量
	int i = 0;
	int* const ptr1 = &i;
	// 5. 底层const: 指针所指的对象是一个常量。等价于常量指针
	int ci = 42;
	const int* ptr2 = &ci;
```
